World Wide Web

Web Develpoment technologies, references & resources

HyperText Markup Language (HTML) is the most basic building block of the Web. It defines the meaning and structure of web content.

Cascading Style Sheets (CSS) is a stylesheet language used to describes how elements should be rendered on screen, on paper, in speech, or on other media.

JavaScript (JS) is a lightweight, interpreted, or just-in-time compiled programming language with first-class functions, which adds dynamic interactivity to websites.
